# 📚 User Guide — PyLess

## Quick Start

1. Open **https://your-domain.com**
2. Click **"Продовжити без реєстрації"** (continue as guest) or register
3. Choose a lesson from the sidebar
4. Write code in the editor and click **▶ Запустити**
5. Click **✅ Тести** to check your solution

## Account

### Registration
- Click **"Реєстрація"** tab
- Enter username, email, password
- Click **"Створити акаунт"**

### Login
- Click **"Увійти"** tab
- Enter email/username and password
- Or click **"Увійти через Google"** (if configured)

### Guest Mode
- Click **"Продовжити без реєстрації"**
- Your progress is tracked locally
- Register later to sync progress

## Editor

### Split View
- **Left pane**: Task description, hints, expected output
- **Right pane**: Code editor (Monaco) or preview (HTML/CSS/JS)

### Running Code
- **▶ Запустити** — execute your code
- **✅ Тести** — run all test cases
- Output appears in the left pane

### Hints
- Click **"Показати підказку"** to reveal hints one by one
- Hints guide you without giving the answer

### Themes
- Click 🌙/☀️ in header to switch dark/light mode
- Select Monaco theme from dropdown

## Gamification

### XP & Levels
- Complete tasks to earn XP (100/75/50 based on attempts)
- First try bonus: +25 XP
- Difficulty multiplier: ×1.5 for hard tasks

### Streaks
- Solve at least 1 task daily to maintain streak
- Streak multiplier: ×1.5 at 7 days, ×2 at 14 days, ×3 at 30 days
- Grace day: miss 1 day without losing streak

### Combo
- Consecutive correct solutions increase combo
- Higher combo = higher XP multiplier

### Badges
- Earn badges by completing achievements
- Common → Uncommon → Rare → Epic → Legendary

### Daily Challenge
- New challenge every day at midnight
- Bonus XP for completion
- Countdown timer in sidebar

## AI Tutor

- Click **"AI"** button in header
- Ask questions about your code
- Get hints without spoilers
- Powered by Groq (llama-3.3-70b-versatile)

## Video Lessons

- Click **"🎥 Уроки"** in header
- Join active video rooms
- Participate in live lessons with screen sharing

## Progress

- Track your progress in the sidebar
- Each module shows completion percentage
- Profile popup shows stats, badges, leaderboard

## Mobile

- Responsive design works on phones
- Best experience on desktop/laptop
